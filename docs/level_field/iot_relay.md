# ESP relay

This is a extension from the IoT Multisensor

This ESP is used for switching relays and detecting motion.

## Parts list

### DC/DC

12 V -> 5 V `MP1584EN`

- DC Input: 4,5-28V max 3A
- DC Output: 0,8-20V

### PIR

`AM312`

- 2,7 - 12V
